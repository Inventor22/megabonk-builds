# Megabonk Build Reference

Simple web page for viewing all Megabonk builds in one place.

It puts every character build on a single page so you can quickly browse, search, and jump between them without opening a bunch of separate guide pages.

Open [megabonk-builds.html](megabonk-builds.html) in a browser to use it.

Features:

- Search by character, build, weapon, tome, or item.
- Quick jump bar for fast character navigation.
- Direct links back to the original guide pages.
